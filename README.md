Aura – Aplicativo de Venda de Cosméticos

1. Sobre o Projeto
O Aura é um aplicativo desenvolvido para facilitar a venda de cosméticos através de uma plataforma moderna, objetiva e fácil de usar. O foco principal é proporcionar uma experiência de compra rápida, simples e eficiente.

2. Objetivo
Fornecer uma aplicação prática que permita ao usuário navegar entre produtos, visualizar detalhes e realizar compras de maneira intuitiva, com o mínimo de etapas e o máximo de clareza.

3. Funcionalidades
Catálogo de produtos organizado por categorias
Visualização de imagens, preços e descrições detalhadas
Navegação simples com interface moderna
Estrutura pensada para compras rápidas e diretas
Layout intuitivo, ideal para todos os tipos de usuários

4. Experiência do Usuário
O design do Aura foi criado pensando em praticidade. Cada tela foi planejada para reduzir etapas desnecessárias, evitar confusão e facilitar o acesso às informações. A experiência geral é objetiva e agradável.

5. Tecnologias Utilizadas
React Native
Expo
JavaScript
Outras ferramentas e bibliotecas de suporte

6. Status do Projeto
O projeto está em desenvolvimento, com novas funcionalidades sendo adicionadas e melhorias contínuas em andamento.
